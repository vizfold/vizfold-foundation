OpenFold Attention Web Prototype
================================

Overview
--------

Streamlit dashboard for exploring saved attention renders. The app expects PNG assets generated by the demo utilities.

Quickstart
----------

```bash
pip install streamlit==1.39
streamlit run webapp/app.py --server.port 8501
```

Environment variable `ATTENTION_OUTPUT_ROOT` can override the default search path (`outputs/`). Use the sidebar to browse attention type, protein, layer, head, and (for triangle start) residue indices. The UI displays available 3D renders, arc diagrams, and combined panels side by side.
